# 🔍 ML Model Comparison App (Streamlit)

An interactive Machine Learning Model Comparison App built using Streamlit and Scikit-learn.
This app allows users to upload structured CSV datasets and compare multiple algorithms for both regression and classification tasks.

## 📊 Features

- Supports both Regression and Classification problems
- Implements 12 Machine Learning algorithms
- Automatic train-test splitting
- R² score evaluation for regression models
- Accuracy evaluation for classification models
- Dynamically selects the best-performing model
- Interactive web interface using Streamlit

---

## 📈 Regression Models Implemented

- Multiple Linear Regression
- Polynomial Regression
- Support Vector Regression (SVR)
- Decision Tree Regression
- Random Forest Regression

---

## 📊 Classification Models Implemented

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Linear SVM
- Kernel SVM (RBF)
- Naive Bayes
- Decision Tree Classification
- Random Forest Classification

---

## 🧠 Key Learnings

- Importance of consistent train-test splitting
- Effect of dataset ordering on model performance
- Proper preprocessing and feature scaling
- Model variance on small datasets
- Basics of ML model deployment using Streamlit

---

## 📂 Supported Dataset Format

The app currently supports structured tabular datasets similar to:

- Salary_Data (Regression)
- Social_Network_Ads (Classification)

The last column is treated as the target variable.
